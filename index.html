Programming projects implemented in every* language

*well, many

What this is about

Using the same programming project across many different languages allows you to concentrate on how a particular language is used, rather than on solving a problem.

This idea was picked up from Turing Incomplete's Justin Campbell.

This organization is open to anyone to contribute. If you would like to contribute, create an issue and I'll make a repo for you and give you full access to that repo. And if you'd like to contribute an implementation to any of the projects below, submit a pull request to that repo.

Ready Projects

We do not currently have any projects ready for you to look at. Ready projects are those that have multiple languages implemented, and are building up additional languages, and open for contributions in a language that is not currently covered

In Development

In Development projects are ones that aren't quite ready for public consumption, but that doesn't mean code doesn't exist. Either they aren't implemented, or there is only a single reference implementation.
